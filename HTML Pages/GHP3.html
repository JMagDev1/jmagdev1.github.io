Google Home Page 3

On the left is the custom layout of links, here the user can display up to 20 links of their choice.

This image showcases the layout of preset 1, this is the layout I copied over from my 2nd Google Home Page. Using this preset the user cannot add or remove any links in the settings menu. Your preference on layout is saved using localstorage and will re open as that layout when the page is closed.

The image on the left shows the settings bar along the bottom of the screen. This is used to: Set your name, save links to the button container on the page, Remove links, adjust the blur and brightness of the image, setting the background picture theme and also set the colour theme of the page. The name and links are saved using localstorage and so does the background theme and colour theme.

This image shows the settings bar for those who choose the preset for links on the Google Home Page. As you can see you can't add or remove any buttons with this preset because it is a set layout and buttons. This can be beneficial for the user because they don't need to spend time adding custom links.
