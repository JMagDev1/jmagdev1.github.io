Marvel Movies Project

This menu page is what the user will see when they start the application and they will be greeted with two options. To either view the list of movies in timeline order or by their order in each phase.

If the user selects the phase order button, they will be greeted with this page, it will ask them what phase they want to view the movies in and they are able to select between the 4 phases.

As an example I have selected to watch the movies in phase three, this is an example of what the user would see if they selected phase three.

If the user selects the Timeline order on the main menu, this is the sreen they are presented with, it gives them all of the movies in the order they are intended to be watched(not the release date).

When the user selects a movie from either menu they will be greeted with a similar menu to this picture(on the right). It gives the user a brief overview of the film and there is a button aallowing the user to view the trailer on youtube.
